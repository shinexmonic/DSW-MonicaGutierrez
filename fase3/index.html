HTML

HTML (Lenguaje de Marcas de Hipertexto, del inglés HyperText Markup Language) es el componente más básico para la construcción de paginas web, generalmente se utilizan otras tecnologías que lo acompañan para describir su diseño o presentación conocido como CSS, y para contenido dinámico o funcionalidad JavaScript.


"Hipertexto" hace referencia a los enlaces que conectan páginas web entre sí, ya sea internamente o dentro de nuestro sitio web u otros externos. Los enlaces son un aspecto fundamental de la Web ya que podemos vincular mas contenido que abunda en la web y ayudamos a la información mundial.


HTML utiliza "elementos" para agregar contenido en forma de texto, imágenes y otro contenido para mostrarlo en un navegador Web, etiquetas que se mostrarán más adelante


Un elemento HTML se distingue de otro texto en un documento mediante "etiquetas", que consisten en el nombre del elemento rodeado por "<" y ">".


Propósito del sitio

El objeto de este sitio es informar al usuario sobre las diferentes etiquetas HTML, las mas importantes, cuando usarlas, para que, dando una breve descripción de estas se logre captar un mayor aprendizaje a traves de ejemplos y consultando sus atributos mínimos.
